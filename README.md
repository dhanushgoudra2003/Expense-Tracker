# Expense Tracker

A simple and efficient web application for tracking your daily expenses and managing your budget effectively.

## ✨ Features

- **Track Expenses**: Easily add, edit, and delete expense entries
- **Categorize Spending**: Organize expenses by different categories
- **View Summary**: Get a quick overview of your spending patterns
- **Budget Management**: Set budget limits and monitor your spending
- **Data Persistence**: Local storage to keep your data safe
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **User-Friendly Interface**: Intuitive and easy-to-use dashboard

## 🛠️ Tech Stack

- **Frontend**: JavaScript (93.5%)
- **Markup**: HTML (4.6%)
- **Styling**: CSS (1.9%)
- **Storage**: Browser Local Storage
- **Version Control**: Git & GitHub

## 📦 Installation

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Git installed on your system

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/dhanushgoudra2003/Expense-Tracker.git
   cd Expense-Tracker
   ```

2. **Open the application**
   - Simply open the `index.html` file in your web browser
   - Or use a local server:
     ```bash
     # Using Python
     python -m http.server 8000
     
     # Using Node.js
     npx http-server
     ```

3. **Access the app**
   - Open your browser and navigate to `http://localhost:8000`

## 🚀 Usage

### Adding an Expense
1. Fill in the expense amount
2. Select a category
3. Add a description (optional)
4. Click "Add Expense" button

### Viewing Expenses
- View all your expenses in the dashboard
- Check the total spending at a glance
- Filter expenses by category

### Editing an Expense
1. Click on an expense entry
2. Modify the details
3. Save the changes

### Deleting an Expense
- Click the delete button next to an expense to remove it

### Setting a Budget
1. Navigate to the budget settings
2. Enter your monthly budget limit
3. Save the settings
4. Monitor your spending against the budget
